===============================================================

# Download the Game:

git clone https://github.com/AnonAmosAdmn/tetris.py.git

===============================================================

# Create a Virtual Environment:

python -m venv mygame_env

source mygame_env/bin/activate  # Linux/macOS

mygame_env\Scripts\activate     # Windows

===============================================================

# Then install Pygame:

pip install pygame

===============================================================

# Run the game :

python3 tetris.py

===============================================================
